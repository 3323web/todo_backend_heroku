# Study Todo Backend

Node.js 기반의 Todo 애플리케이션 백엔드입니다.

## 시작하기

### 필수 요구사항
- Node.js (v18 이상 권장)

### 설치 및 실행

1. 프로젝트 의존성 설치:
```bash
npm install
```

2. 개발 모드로 실행:
```bash
npm run dev
```

3. 프로덕션 모드로 실행:
```bash
npm start
```

## 프로젝트 구조

```
study_todo_backend/
├── index.js             # 애플리케이션 진입점
├── package.json         # 프로젝트 설정 및 의존성
├── .gitignore          # Git 무시 파일 목록
└── README.md           # 프로젝트 문서
```

## 환경 변수

프로젝트 루트에 `.env` 파일을 생성하여 환경 변수를 설정할 수 있습니다.

예시:
```
PORT=3000
NODE_ENV=development
```
